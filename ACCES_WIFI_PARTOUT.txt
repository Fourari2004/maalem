📦 OPTION 1: NGROK (Accès temporaire - Gratuit)
═══════════════════════════════════════════════════════════════════

✅ AVANTAGES:
   - Configuration en 5 minutes
   - Gratuit pour commencer
   - Fonctionne depuis n'importe où dans le monde
   - HTTPS automatique

❌ INCONVÉNIENTS:
   - URL change à chaque redémarrage (version gratuite)
   - Limité à 8 heures de session
   - Bannière ngrok affichée
   - Votre ordinateur doit rester allumé

┌──────────────────────────────────────────────────────────────────┐
│ ÉTAPES RAPIDES:                                                  │
└──────────────────────────────────────────────────────────────────┘

1. INSTALLER NGROK
   → Allez sur: https://ngrok.com/download
   → Téléchargez pour Windows
   → Créez un compte gratuit
   → Extrayez ngrok.exe dans un dossier (ex: C:\ngrok\)

2. CONFIGURER NGROK
   → Allez sur: https://dashboard.ngrok.com/get-started/your-authtoken
   → Copiez votre authtoken
   → Dans PowerShell:
     C:\ngrok\ngrok.exe config add-authtoken VOTRE_TOKEN

3. LANCER AVEC LE SCRIPT
   → Double-cliquez sur: start_ngrok.bat
   
   OU manuellement:
   
   Terminal 1: cd maalem-backend && python manage.py runserver 0.0.0.0:8000
   Terminal 2: cd maalem-frontend && npm run dev -- --host
   Terminal 3: ngrok http 8000
   Terminal 4: ngrok http 5173

4. NOTER LES URLs NGROK
   Dans les terminaux ngrok, vous verrez:
   → Backend:  https://xxxx-1234.ngrok-free.app
   → Frontend: https://yyyy-5678.ngrok-free.app

5. METTRE À JOUR LE FRONTEND
   → Créez: maalem-frontend\.env.ngrok
   → Contenu: VITE_API_URL=https://xxxx-1234.ngrok-free.app/api
   → Copiez: copy .env.ngrok .env
   → Redémarrez le frontend (Terminal 2)

6. PARTAGER!
   → Donnez l'URL frontend à qui vous voulez
   → Ils peuvent accéder depuis n'importe quel appareil/WiFi

🔒 SÉCURITÉ - IMPORTANT!
═══════════════════════════════════════════════════════════════════

Avant de partager publiquement:

1. ✅ Changez le SECRET_KEY dans settings.py
2. ✅ Utilisez des mots de passe forts pour tous les comptes
3. ✅ Activez HTTPS (ngrok le fait automatiquement)
4. ✅ Ne partagez jamais votre authtoken ngrok
5. ✅ Limitez les permissions utilisateurs
6. ✅ Activez l'authentification à deux facteurs si disponible

✅ CE QUI SE LANCE AUTOMATIQUEMENT

Le script fait TOUT pour vous:

✓ Détecte automatiquement votre adresse IP
✓ Configure le frontend pour l'accès mobile
✓ Lance le backend Django (port 8000)
✓ Lance le frontend Vite (port 5173)
✓ Configure pour l'accès réseau local
✓ Ouvre le site dans votre navigateur

═══════════════════════════════════════════════════════════════════

🖥️ ACCÈS DEPUIS VOTRE ORDINATEUR

Frontend: http://localhost:5173
Backend:  http://localhost:8000

═══════════════════════════════════════════════════════════════════

📱 ACCÈS DEPUIS VOTRE TÉLÉPHONE

1. Connectez votre téléphone au MÊME WiFi que votre ordinateur

2. L'adresse sera affichée dans le script (exemple):
   http://192.168.68.58:5173

3. Ouvrez cette adresse sur votre téléphone

4. Connectez-vous normalement!

❓ PROBLÈMES COURANTS

Problème: "Python n'est pas reconnu"
Solution: Installez Python et ajoutez-le au PATH

Problème: "npm n'est pas reconnu"
Solution: Installez Node.js

Problème: Le téléphone ne se connecte pas
Solution: 
  1. Vérifiez que vous êtes sur le même WiFi
  2. Lancez configure_firewall.ps1 (administrateur)
  3. Vérifiez l'IP affichée dans le script

Problème: Port déjà utilisé
Solution: Fermez les autres serveurs ou redémarrez l'ordinateur
